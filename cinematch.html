Overview

CineMatch is a search and recommendation engine for TV series built from subtitle file analysis of 128 series. The system uses natural language processing and text mining techniques to provide intelligent search and personalized recommendations based on textual content rather than metadata.

The Challenge

Traditional series recommendation systems rely on genre, actors, or user ratings. The challenge was to create a content-based system that understands what a series is actually about by analyzing its dialogue and narrative content through subtitle files.

Technical Implementation

Data Processing Pipeline:

  • Text cleaning and preprocessing of subtitle files
  • Removal of irrelevant words and noise
  • Database indexing for fast query performance
  • TF-IDF vectorization for content representation

Search Engine: Keyword-based retrieval using text mining principles with optimized response times under 0.5 seconds.

Recommendation System: Content-based filtering using user ratings (1-5 scale) combined with textual similarity to suggest relevant series.

REST API & Web Interface

Developed a complete REST API exposing search and recommendation endpoints, with an ergonomic web interface allowing users to:

  • Search series using natural language keywords
  • Rate series from 1 to 5 stars
  • Receive personalized recommendations based on viewing history
  • Explore series content through semantic understanding

Key Achievements

  • High Relevance: Top-3 accuracy for semantic queries
  • Fast Performance: Sub-second response times for all queries
  • Scalable Architecture: Clean API design for future expansion
  • User-Centric Design: Intuitive interface for seamless interaction
